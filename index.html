Meus projetos

Torneio Juvenil de Robótica

A equipe participou de uma competição de robótica, onde o robô foi desenvolvido utilizando a plataforma LEGO Mindstorms. Eles se tornaram campeões na etapa nacional e, com isso, garantiram a classificação para a etapa internacional

Programação em JAVA

Desenvolvemos um robô para a competição FRC usando a linguagem de programação Java. O robô é capaz de se movimentar com precisão através do chassi, alcançar diferentes alturas com um elevador e coletar e ejetar game pieces de forma eficiente. Utilizamos a biblioteca WPILib e sensores para garantir um controle preciso do robô, otimizando seu desempenho durante as partidas.

Programação em LabVIEW

Utilizamos o LabVIEW para criar um robô FRC com controle visual intuitivo, permitindo manobras precisas com o chassi, elevação versátil para alcançar diferentes alturas e um sistema eficiente de coleta e ejeção de game pieces. Sensores e atuadores são integrados para otimizar o desempenho do robô, com comunicação robusta entre o robô e o driver station, tudo facilitado pelas bibliotecas FRC do LabVIEW.
